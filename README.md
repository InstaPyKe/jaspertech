# Jasper-Brevin Technologies Portfolio

A high-end, professional portfolio website for a Full-Stack Developer and Security Consultant. This project features a modern Glassmorphism UI, interactive physics-based background animations, and comprehensive SEO optimization.

## 🚀 Features

- **Glassmorphism Design**: High-fidelity UI using backdrop filters and neon shadows.
- **Interactive Elements**: Physics-based balloon animation canvas.
- **SEO Optimized**: Fully mapped with Meta tags, Open Graph (OG) tags, and JSON-LD Structured Data.
- **Responsive UI**: Mobile-first design with a custom hamburger menu and staggered fade-in animations.
- **Typewriter Effect**: Dynamic hero sub-headlines.
- **Secure Contact Form**: Professional form with success state overlays.

## 🛠️ Tech Stack

- HTML5 / CSS3 (Modern Flexbox & Grid)
- Vanilla JavaScript (Intersection Observer, Canvas API)
- Google Fonts (Inter)
